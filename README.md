# 🎓 AI Student Performance Analytics Platform

An AI-powered web application that predicts student math scores and provides interactive analytics dashboards.

## 🚀 Features

- Student score prediction using Machine Learning
- Interactive Streamlit dashboard
- Data visualization and analytics
- User authentication system
- Prediction history tracking
- Download prediction results

## 🛠 Tech Stack

- Python
- Streamlit
- Scikit-learn
- Pandas
- Plotly
- SQLite

## 📊 Machine Learning Model

The system predicts **Math Score** based on:

- Reading Score
- Writing Score
- Gender
- Lunch Type

Models used:
- Linear Regression
- Decision Tree
- Random Forest
